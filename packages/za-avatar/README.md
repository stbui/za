# Avatar

<carbon-ad></carbon-ad>

## Installation

```sh
npm install za-avatar
```

## Usage

```jsx
import Avatar from 'za-avatar';

function Example() {
    return (
        <>
            <Avatar
                src="images/avatars/blackwidow.png"
                size="xsmall"
                name="Natalia Alianovna Romanova"
            />
        </>
    );
}
```
