# 快速开始

stbui 是一个低级组件库，用于构建可访问的高级 ui 库、设计系统和带有 react 的应用程序。

## 安装

```base
npm install @stbui/za
```

### 基本使用

```js
import Button from '@stbui/za-button';

function Example() {
    return (
        <Button style={{ color: 'white', background: 'red' }}>Button</Button>
    );
}
```
