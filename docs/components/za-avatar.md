# Avatar

## Installation

```sh
npm install @stbui/za-avatar
```

## Usage

```jsx
import React from 'react';
import Avatar from '@stbui/za-avatar';

export default () => (
    <Avatar
        src="images/avatars/blackwidow.png"
        size="xsmall"
        name="Natalia Alianovna Romanova"
    />
);
```
