# tag

## Installation

```sh
npm install @stbui/za-tag
```

## Usage

```jsx
import React from 'react';
import Tag from '@stbui/za-tag';

export default () => (
    <>
        <Tag>Tag</Tag>
    </>
);
```

## API

| 参数    | 说明         | 类型    | 默认值 | 版本 |
| :------ | :----------- | :------ | :----- | :--- |
| onClose | 关闭时的回调 | boolean | false  |      |
